ANGRY INTERN

Platform: Windows | Engine: Unity C# | Role: Lead Programmer

Angry Intern is a stealth-based game where the player must sabotage their company by causing as much havoc as possible, which involves various interactions between various objects found in the office.

As Lead Programmer, I worked with the designers to develop and implement game software, and ensured game design was fully implemented and performed as expected.

  • Designed and balanced the system of object interactions
  • Programmed an inventory system and UI to use collected items
  • Created basic AI pathfinding
  • Responsible for programming dynamic audio to change depending on several variables
